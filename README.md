# local-ai-friend
Гибридный локальный ИИ-помощник, настроенный на стыке Senior-разработки, прагматичного психоанализа и живого дружеского общения. Работает полностью приватно на базе **Ollama** и модели **Qwen 2.5**, интегрируется прямо во «второй мозг» на базе **Obsidian**.

---

## 🛠️ Архитектура и фичи

1. **Senior Developer Mode**: Автоматически включается на код, логи или дедлайны. Выдает чистые скрипты моментально, без лишней вежливости и воды. Основной стек: `Python` (автоматизация, Termux, прокси) и `C++` (сетевой стек, Android NDK, HyperOS/MTK прошивки).
2. **Прагматичный психолог**: Анализирует выгорание, усталость и затыки с дедлайнами через логику и деконструкцию эмоций. Никакой банальной мотивации — только разбор полетов.
3. **Автономная долгосрочная память**: В конце каждого диалога генерирует скрытый тег `:::memory` для автоматической сквозной индексации через плагин *Smart Connections*. Через неделю чатов ИИ будет знать контекст вашей жизни.
4. **Вайб**: Разговор двух коллег-разработчиков после тяжелого дня. На «ты», с легкой иронией.

## 🚀 Быстрый старт (Deployment)

### 1. Подготовка движка (Ollama)
Установите Ollama и скачайте текстовую модель от Alibaba:
```bash
# Скачивание движка через WinGet
winget install Ollama.Ollama

# Скачивание модели (требуется ~4.7 ГБ)
ollama run qwen2.5:7b
```

### 2. Оптимизация под скорость
В настройках инстанса модели обязательно выставьте параметры для быстрой генерации на домашнем ПК:
* `Token limit`: `16000`
* `Temperature`: `0.75` (для живой речи и гибкости мышления)
* `num_ctx`: `8192` или `16384` (не ставьте больше, иначе у процессора закипит мозг).

### 3. Интеграция в Obsidian
1. Установите комьюнити-плагин **Copilot** (автор *Logico*).
2. В настройках укажите:
   * **Provider**: `Ollama`
   * **Ollama URL**: `http://localhost:11434`
   * **Active Model**: `qwen2.5:7b`
3. Скопируйте системный промпт из файла `sputnik_prompt.txt` данного репозитория и вставьте его во вкладке **Advanced** -> **Default System Prompt**.
4. Включите тумблеры `Autosave Chat` и `Auto-Add Active Content to Context`.
