# local-ai-friend
Локальный ИИ в Obsidian запускается через связку Ollama (которая крутит нейросеть на твоем ПК) и плагина Copilot (который выводит минималистичный чат на панель заметок).

# Шаг 1. Проверяем движок Ollama
Убедись, что Ollama запущена в системе (в трее Windows около часов должен гореть значок белого ламы).Если ты ещё не скачал модель Qwen 2.5, открой обычную командную строку Windows (cmd) и введи:
```
ollama run qwen2.5:7b
```
* Зависит от производительности ПК
